JavaScript Strings & String Methods

Deep explanation with examples, compiled in a clean and minimal website format.

1. What is a String in JavaScript?

  • A string is a sequence of characters enclosed in quotes.
  • Strings are immutable – once created, you can’t change characters inside it; you create a new string.
  • You can use 'single', "double", or `backticks` (template literals). 
let str1 = "Hello";    // double quotes
let str2 = 'World';    // single quotes
let str3 = `Hello ${str2}`; // template literal
console.log(str3); // Hello World

2. How Strings are Stored

  • A string is array-like (you can access characters by index), but not an actual array.
  • Indices start at 0.
let word = "JavaScript";
console.log(word[0]);  // J
console.log(word[4]);  // S
console.log(word.length); // 10

3. String Methods in JavaScript

3.1 length (Property)

Gives the number of characters.

let txt = "Hello World";
console.log(txt.length); // 11

3.2 Case Conversion

  • .toUpperCase() → Converts to UPPERCASE.
  • .toLowerCase() → Converts to lowercase.
let msg = "JavaScript";
console.log(msg.toUpperCase()); // "JAVASCRIPT"
console.log(msg.toLowerCase()); // "javascript"

3.3 Searching Methods

  • .indexOf(substring)
  • .lastIndexOf(substring)
  • .includes(substring)
  • .startsWith(substring)
  • .endsWith(substring)
let sentence = "I love JavaScript";
console.log(sentence.indexOf("love"));  // 2
console.log(sentence.includes("Java")); // true
console.log(sentence.startsWith("I"));  // true
console.log(sentence.endsWith("pt"));   // true

3.4 Extracting Part of a String

let str = "Hello World";
console.log(str.slice(0, 5));      // "Hello"
console.log(str.substring(6, 11)); // "World"
console.log(str.substr(6, 5));     // "World"

3.5 Replacing

let text = "I love JavaScript. JavaScript is fun.";
console.log(text.replace("JavaScript", "JS"));
console.log(text.replaceAll("JavaScript", "JS"));

3.6 Removing Extra Spaces

let messy = "   Hello JS   ";
console.log(messy.trim());
console.log(messy.trimStart());
console.log(messy.trimEnd());

3.7 Splitting & Joining

let data = "apple,banana,grape";
let arr = data.split(",");
console.log(arr); // ["apple", "banana", "grape"]

3.8 Repeating

console.log("Hi!".repeat(3)); // "Hi!Hi!Hi!"

3.9 Character Access

let s = "ABC";
console.log(s.charAt(1));     // "B"
console.log(s.charCodeAt(1)); // 66

3.10 Concatenation

let a = "Hello";
let b = "World";
console.log(a.concat(" ", b));
console.log(a + " " + b);
console.log(`${a} ${b}`);

3.11 Padding

let num = "5";
console.log(num.padStart(3, "0"));
console.log(num.padEnd(3, "*"));

3.12 Template Literals

let name = "Sajid";
let age = 21;
console.log(`My name is ${name} and I am ${age} years old.`);

4. Key Things to Remember

  1. Strings are immutable.
  2. Use template literals for readability.
  3. slice is more powerful than substring.
  4. includes is simpler than indexOf.
  5. Always trim input before saving.
