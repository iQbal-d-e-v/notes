JavaScript Operators

In this page, we are going to learn about different types of operators in JavaScript, how they work, and see examples for each one. Operators are special symbols used to perform operations on values and variables.

1. Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations like addition, subtraction, multiplication, division, modulus (remainder), and exponentiation. 


let a = 10, b = 3;
console.log(a + b); // 13 (Addition)
console.log(a - b); // 7  (Subtraction)
console.log(a * b); // 30 (Multiplication)
console.log(a / b); // 3.33 (Division)
console.log(a % b); // 1  (Remainder)
console.log(a ** b); // 1000 (Exponentiation)

2. Assignment Operators

Assignment operators are used to assign values to variables. They can also perform operations and assign the result in one step. 


let x = 10;
x += 5; // x = x + 5 → 15
x -= 3; // x = x - 3 → 12
x *= 2; // x = x * 2 → 24
x /= 4; // x = x / 4 → 6
x %= 5; // x = x % 5 → 1

3. Comparison Operators

Comparison operators compare two values and return true or false. They are used in conditions. 


console.log(5 == "5");  // true  (value equal, type ignored)
console.log(5 === "5"); // false (strict equality, type checked)
console.log(7 != 8);    // true  (not equal)
console.log(7 !== "7"); // true  (strict not equal)
console.log(10 > 6);    // true
console.log(4 <= 2);    // false

4. Logical Operators

Logical operators are used to combine multiple conditions: AND (&&), OR (||), and NOT (!). 


console.log(true && false); // false (both must be true)
console.log(true || false); // true  (at least one true)
console.log(!true);         // false (NOT reverses value)

5. Increment and Decrement

These operators increase or decrease a value by 1. They can be used in two ways: pre (++x, --x) and post (x++, x--). 


let y = 5;
console.log(y++); // 5 (returns then increments → y=6)
console.log(++y); // 7 (increments then returns)

6. Ternary Operator

A shorthand for if...else. It checks a condition and returns one value if true, and another if false. 


let age = 20;
let result = (age >= 18) ? "Adult" : "Minor";
console.log(result); // Adult

7. Bitwise Operators

These operators work at the binary (bit) level: & (AND), | (OR), ^ (XOR), ~ (NOT), << (left shift), >> (right shift). 


console.log(5 & 3);  // 1  (0101 & 0011 = 0001)
console.log(5 | 3);  // 7  (0101 | 0011 = 0111)
console.log(5 ^ 3);  // 6  (0101 ^ 0011 = 0110)
console.log(~5);     // -6 (bitwise NOT)
console.log(5 << 1); // 10 (shift left)
console.log(5 >> 1); // 2  (shift right)

8. Nullish Coalescing (??)

The nullish coalescing operator ?? returns the right-hand value only if the left-hand value is null or undefined. 


let name = null;
console.log(name ?? "Guest"); // Guest

9. Optional Chaining (?.)

Optional chaining prevents errors when accessing nested object properties that might not exist. Instead of crashing, it returns undefined. 


let user = { profile: { name: "Alex" } };
console.log(user.profile?.name);   // Alex
console.log(user.contact?.email);  // undefined (safe, no error)
