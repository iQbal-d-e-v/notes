ðŸ”¹ What is a Function?

A function is a block of code designed to perform a particular task. Instead of repeating the same code, we can wrap it in a function and reuse it whenever needed. Functions improve reusability, readability, and organization.

ðŸ”¹ Basic Syntax

function functionName(parameters) {
  // code to be executed
  return value; // optional
}

- function keyword declares the function.
- functionName is the name you give.
- parameters are placeholders for input values.
- return sends back a result (optional).

ðŸ”¹ Example 1: Simple Function

function greet() {
  console.log("Hello, Welcome to JavaScript!");
}

greet(); // calling the function

This function has no parameters and simply prints a welcome message. To execute it, we call it using greet().

ðŸ”¹ Example 2: Function with Parameters

function greetUser(name) {
  console.log("Hello " + name + ", welcome to JavaScript!");
}

greetUser("Sajid");
greetUser("Ali");

Here, name is a parameter. When calling the function, we pass arguments like "Sajid" and "Ali". Each call produces personalized output.

ðŸ”¹ Example 3: Function with Return Value

function add(a, b) {
  return a + b;
}

let result = add(5, 7);
console.log("Sum is:", result);

This function takes two parameters a and b, adds them, and returns the result. Without return, functions give undefined.

ðŸ”¹ Types of Functions

1. Function Declaration

function sayHello() {
  console.log("Hello!");
}
sayHello();

2. Function Expression

const sayBye = function() {
  console.log("Goodbye!");
};
sayBye();

3. Arrow Function

const multiply = (x, y) => x * y;
console.log(multiply(4, 5));

4. Anonymous Function

setTimeout(function() {
  console.log("Runs after 2 seconds");
}, 2000);

5. IIFE (Immediately Invoked Function Expression)

(function() {
  console.log("IIFE executed!");
})();

ðŸ”¹ Function Parameters

Default Parameters

function greet(name = "Guest") {
  console.log("Hello " + name);
}

greet("Sajid");
greet();

Rest Parameters

function sum(...numbers) {
  return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(1,2,3,4,5));

ðŸ”¹ Function Scope

let globalVar = "I am global";

function checkScope() {
  let localVar = "I am local";
  console.log(globalVar); // works
  console.log(localVar); // works
}

checkScope();
console.log(localVar); // Error

- Variables declared inside a function are local.
- Variables declared outside are global.
- Local variables are not accessible outside.

ðŸ”¹ Higher Order Functions

function operate(a, b, operation) {
  return operation(a, b);
}

let result = operate(5, 3, function(x, y) { return x + y; });
console.log(result);

Functions can be passed as arguments or returned. These are called higher-order functions.

  if (n === 0) return 1;
