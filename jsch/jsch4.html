JavaScript Conditional Statements

Conditional statements in JavaScript are the foundation of decision-making in programming. They allow your code to perform different actions depending on whether a condition is true or false. In this tutorial, we will cover all types of conditional statements in detail with examples and best practices.

1. What are Conditional Statements?

Conditional statements help us control the flow of a program. They are used when we want to execute certain parts of code only if a particular condition is satisfied. 

// Example
let age = 18;

if (age >= 18) {
  console.log("You are an adult.");
}

Here, the message will only be displayed if the condition age >= 18 is true.

2. The if Statement

The if statement checks whether a condition is true. If the condition evaluates to true, the block of code inside it executes. 

let number = 10;

if (number > 0) {
  console.log("The number is positive.");
}

Explanation: Since 10 > 0 is true, the program outputs The number is positive.

3. The else Statement

The else statement provides an alternative path. If the if condition is false, the else block executes. 

let number = -5;

if (number > 0) {
  console.log("Positive number");
} else {
  console.log("Not a positive number");
}

Here, since -5 > 0 is false, the program executes the else block and prints Not a positive number.

4. The else if Statement

The else if statement is used to check multiple conditions. It prevents the need for writing multiple separate if blocks. 

let marks = 75;

if (marks >= 90) {
  console.log("Grade A");
} else if (marks >= 75) {
  console.log("Grade B");
} else {
  console.log("Grade C");
}

Explanation: The code checks conditions in order. Since marks = 75 matches the else if condition, it prints Grade B.

5. The switch Statement

A switch statement is often more elegant than multiple else if conditions when comparing the same variable. 

let fruit = "apple";

switch(fruit) {
  case "apple":
    console.log("Apples are red or green.");
    break;
  case "banana":
    console.log("Bananas are yellow.");
    break;
  default:
    console.log("Unknown fruit.");
}

The switch checks the value of fruit and executes the matching case. The break statement stops the switch from continuing further.

6. Ternary Operator

The ternary operator is a shorthand for simple if-else conditions. It uses the syntax: condition ? valueIfTrue : valueIfFalse. 

let age = 20;

let message = (age >= 18) ? "Adult" : "Minor";
console.log(message);

Since age = 20, the condition is true, and the output will be Adult.

7. Nested Conditionals

Conditionals can be nested inside each other, but readability should be considered carefully. Too much nesting can make code hard to understand. 

let age = 25;

if (age > 18) {
  if (age > 21) {
    console.log("You are above 21");
  } else {
    console.log("You are between 18 and 21");
  }
}

Here, the program first checks if age > 18. If true, it goes further to check if age > 21.
