Loops in JavaScript

On this page, we will explore loops in JavaScript. Loops are a way to repeatedly execute blocks of code, either a fixed number of times or until a condition is met. You’ll learn different types of loops, when to use each one, and practical examples with detailed explanations.

1. The Classic for Loop

The for loop is one of the most commonly used loops. You use it when you know how many times you want to run your code, or when you need access to an index while iterating.

Structure: for (initialization; condition; increment) { ... }

// Example: Sum numbers from 1 to n
function sumTo(n) {
  let sum = 0;
  for (let i = 1; i <= n; i++) {
    sum += i;
  }
  return sum;
}
console.log(sumTo(5)); // 15

Here, i starts at 1, increases by 1 each time, and the loop continues until i is greater than n.

2. while and do...while Loops

These loops repeat code based on a condition. The while loop checks the condition first, while do...while runs the block at least once before checking. 

// while example
let countdown = 3;
while (countdown > 0) {
  console.log(countdown);
  countdown--;
}
console.log("Go!");
// do...while example
let input;
do {
  input = "Sample Input";
  console.log("Processed:", input);
} while (false);

3. The for...of Loop

for...of is used to iterate over values of iterable objects like arrays, strings, sets, or maps. 

const colors = ["red", "green", "blue"];
for (const color of colors) {
  console.log(color);
}

This prints each element in the array without needing to deal with indexes.

4. The for...in Loop

for...in is used to iterate over the keys (property names) of an object. 

const person = { name: "Ava", role: "Developer" };
for (const key in person) {
  if (Object.hasOwn(person, key)) {
    console.log(key, person[key]);
  }
}

This ensures we only get the object’s own keys, not those inherited from its prototype.

5. Array Iteration Methods

Modern JavaScript arrays provide built-in methods to loop in a cleaner, more functional style. 

const nums = [1, 2, 3, 4];

// forEach: runs for each element
nums.forEach((n) => console.log(n));

// map: returns a new array
const squares = nums.map(n => n * n);

// filter: keeps only some values
const evens = nums.filter(n => n % 2 === 0);

// reduce: combines values into one
const sum = nums.reduce((acc, n) => acc + n, 0);

Use these when you want to transform arrays or compute results without writing manual loops.

6. Break and Continue

Inside loops, you can use break to stop the loop entirely or continue to skip the current iteration. 

for (let i = 0; i < 10; i++) {
  if (i === 3) continue; // skip 3
  if (i === 7) break;    // stop at 7
  console.log(i);
}

7. Asynchronous Loops

When dealing with promises, you often need to use for...of with await for sequential async operations. 

async function fetchInOrder(urls) {
  const results = [];
  for (const url of urls) {
    const res = await fetch(url);
    results.push(await res.text());
  }
  return results;
}

This ensures each fetch finishes before the next one starts.

8. Quick Cheat Sheet

  • Need an index or break/continue → Use for
  • Iterating over array values or strings → Use for...of
  • Iterating over object keys → Use for...in (or Object.keys/entries)
  • Transforming/filtering arrays → Use map, filter, reduce
  • Sequential async work → Use for...of with await
