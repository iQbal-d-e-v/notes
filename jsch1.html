JavaScript Datatypes with Examples

1. String

A sequence of characters used to represent text. Strings are immutable and written in quotes.

let name = "Sajid";
console.log("Hello, " + name); // Hello, Sajid
console.log(name.length);      // 5

2. Number

Represents integers and floating-point values. Includes special values like Infinity, -Infinity, and NaN.

let a = 10;
let b = 3.5;
console.log(a + b); // 13.5
console.log(a / b); // 2.857142857...

3. Boolean

Represents logical values true or false, used in conditions and logic.

let isLoggedIn = true;
let age = 18;
console.log(age >= 18); // true
console.log(isLoggedIn); // true

4. Null

A special value representing intentional emptiness or no value.

let car = null;
console.log(car);        // null
console.log(typeof car); // "object"

5. Undefined

Indicates a variable has been declared but not assigned a value.

let x;
console.log(x); // undefined

function greet() {}
console.log(greet()); // undefined

6. Symbol

A unique and immutable primitive value often used as object keys.

let id = Symbol("id");
let user = { name: "Sajid", [id]: 123 };
console.log(user.name); // Sajid
console.log(user[id]);  // 123

7. BigInt

Used to represent very large integers beyond the safe limit of Number.

let big = 123456789012345678901234567890n;
console.log(big);       // 123456789012345678901234567890n
console.log(big + 10n); // 123456789012345678901234567900n

8. Object

A collection of key-value pairs where values can be any datatype.

let user = {
  name: "Sajid",
  age: 20,
  isAdmin: true
};

console.log(user.name);    // Sajid
console.log(user["age"]);  // 20

9. Array

A special type of object used to store ordered collections of values.

let fruits = ["apple", "banana", "mango"];
console.log(fruits[0]);      // apple
fruits.push("orange");
console.log(fruits.length);  // 4

10. Function

A block of reusable code that can take inputs (parameters) and return an output.

function add(a, b) {
  return a + b;
}

console.log(add(5, 3)); // 8
