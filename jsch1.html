JavaScript Datatypes

In this page, we will learn about all the fundamental datatypes in JavaScript with detailed explanations and examples. Each section explains the concept clearly and provides sample code for better understanding.

1. String

A String is a sequence of characters used to represent text. Strings can be written using single quotes, double quotes, or backticks. They are immutable, meaning once created, they cannot be changed directly. JavaScript provides many methods for working with strings, such as toUpperCase(), slice(), and includes().

let name = "Sajid";
console.log("Hello, " + name); // Hello, Sajid
console.log(name.length);      // 5

2. Number

The Number type represents both integers and floating-point numbers. JavaScript does not have separate types for integers and decimals. It also supports special values like Infinity, -Infinity, and NaN (Not-a-Number).

let a = 10;
let b = 3.5;
console.log(a + b); // 13.5
console.log(a / b); // 2.857142857...

3. Boolean

A Boolean has only two values: true or false. It is commonly used for decision making in conditions and logical operations. Expressions like comparisons return boolean values.

let isLoggedIn = true;
let age = 18;
console.log(age >= 18); // true
console.log(isLoggedIn); // true

4. Null

Null represents intentional emptiness. It is a special value used to indicate "no value". Unlike undefined, which happens by default, null is usually assigned explicitly.

let car = null;
console.log(car);        // null
console.log(typeof car); // "object" (quirk in JS)

5. Undefined

Undefined means a variable has been declared but no value has been assigned yet. Functions also return undefined if they do not explicitly return a value.

let x;
console.log(x); // undefined

function greet() {}
console.log(greet()); // undefined

6. Symbol

A Symbol is a unique and immutable primitive value introduced in ES6. Even if two symbols have the same description, they are always different. Symbols are often used as object keys to avoid property name conflicts.

let id = Symbol("id");
let user = { name: "Sajid", [id]: 123 };
console.log(user.name); // Sajid
console.log(user[id]);  // 123

7. BigInt

BigInt is a special numeric type used to handle very large integers beyond the safe limit of Number. It is created by appending n to the end of an integer.

let big = 123456789012345678901234567890n;
console.log(big);       // 123456789012345678901234567890n
console.log(big + 10n); // 123456789012345678901234567900n

8. Object

An Object is a collection of key-value pairs. Keys are usually strings (or symbols) and values can be any datatype, including functions or other objects. Objects are the building blocks of JavaScript.

let user = {
  name: "Sajid",
  age: 20,
  isAdmin: true
};

console.log(user.name);    // Sajid
console.log(user["age"]);  // 20

9. Array

An Array is a special type of object used to store ordered collections of values. Arrays are zero-indexed, meaning the first element is at index 0. They come with many useful methods like push(), pop(), map(), and filter().

let fruits = ["apple", "banana", "mango"];
console.log(fruits[0]);      // apple
fruits.push("orange");
console.log(fruits.length);  // 4

10. Function

A Function is a reusable block of code designed to perform a task. It can accept parameters, process them, and return a result. Functions improve modularity and reduce repetition in code.

function add(a, b) {
  return a + b;
}

console.log(add(5, 3)); // 8
