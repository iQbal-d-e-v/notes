1) Variables & Constants

Store and label values. Understand scope rules and reassignment behavior.

  • let: block‑scoped, reassignable.
  • const: block‑scoped, no reassignment (but object contents can change).
  • var: function‑scoped; avoid in modern JS due to hoisting quirks.
let count = 0;
const PI = 3.14159; // cannot reassign PI
var legacy = 'avoid'; // function scoped
Why React cares: Prefer const for values that shouldn't change and let for counters/temp. Predictable scopes reduce bugs in components.

2) Data Types

Know primitives vs references and how they behave.

Primitives: string number boolean null undefined symbol bigint — Reference: object array function 

typeof 'hi' // 'string'
typeof 42   // 'number'
typeof null // 'object' (historical quirk)
Array.isArray([1,2]) // true

3) Operators

Arithmetic & Assignment

let x = 10; x += 5; // 15
x++; // 16, postfix increment

Comparison & Coercion

=== checks value & type; == coerces (avoid unless you know rules).

'2' == 2   // true (coercion)
'2' === 2  // false (strict)

Logical & Ternary

const ok = isReady && hasData;
const title = custom || 'Untitled';
const label = done ? '✅' : '…';

Nullish Coalescing & Logical Assignment

const n = input ?? 0;       // only uses 0 if input is null/undefined
opts.theme ||= 'light';     // sets if falsy
config.cache ??= {};        // sets if null/undefined

4) Strings & Template Literals

Interpolate values and use helpful methods.

const name = 'Sajid';
`Hello, ${name.toUpperCase()}!`;
'  trim  '.trim(); // 'trim'
'hello'.includes('lo'); // true

5) Numbers & Math

Math.round(2.6); // 3
Math.floor(Math.random()*6)+1; // 1..6 dice
Number.parseInt('42', 10); // 42
Number.isNaN(NaN); // true

6) Conditionals

if (score >= 90) grade = 'A';
else if (score >= 75) grade = 'B';
else grade = 'C';

switch (role) {
  case 'admin': allow(); break;
  case 'user': view(); break;
  default: deny();
}

7) Loops

for / while / do..while

for (let i=0;i<3;i++) console.log(i);
let i=0; while(i<3){ console.log(i++); }
let j=0; do { console.log(j++); } while(j<3);

for..of / for..in

for (const n of [10,20,30]) console.log(n);     // values
for (const k in {a:1,b:2}) console.log(k);       // keys

8) Functions

Different ways to declare; parameters, defaults, rest.

// Declaration, hoisted
function add(a,b){ return a+b }

// Expression
const sub = function(a,b){ return a-b }

// Arrow (lexical this)
const mul = (a,b) => a*b;

// Default & Rest
function greet(name = 'Guest'){ return `Hi ${name}` }
function sum(...nums){ return nums.reduce((a,b)=>a+b,0) }

9) Arrays

Common Methods

const arr = [1,2,3,4];
arr.push(5); arr.pop(); arr.shift(); arr.unshift(0);
arr.map(x => x*2);          // [2,4,6,8]
arr.filter(x => x%2===0);   // [2,4]
arr.reduce((a,b)=>a+b,0);   // 10
arr.find(x => x>2);         // 3
arr.some(x => x>3);         // true
arr.every(x => x>0);        // true
arr.forEach(x => console.log(x));
React link: map is used all the time to render lists of JSX.

10) Objects

const user = { id:1, name:'Aisha' };
user.name;        // dot
user['id'];       // bracket
Object.keys(user);    // ['id','name']
Object.values(user);  // [1,'Aisha']
Object.entries(user); // [['id',1],['name','Aisha']]
Object.assign({}, user, {role:'admin'});

11) Destructuring

const { name, id: userId } = user;
const [first, ...rest] = [10,20,30];

12) Spread & Rest

const copy = { ...user, name: 'Sara' };
const arr2 = [...[1,2], 3];
function wrap(...items){ return items }
React state: Use spread to create new objects/arrays instead of mutating existing state.

13) Short‑circuiting & Optional Chaining

const title = settings.title || 'Untitled';
const safeCity = user.address?.city; // no error if address undefined

14) ES Modules (import/export)

// math.js
export const add = (a,b) => a+b;
export default function mul(a,b){ return a*b }

// app.js
import mul, { add } from './math.js';

15) Classes & this

class Person{
  constructor(name){ this.name = name }
  hello(){ return `Hello ${this.name}` }
}

const p = new Person('Noor');
p.hello(); // "Hello Noor"

Arrow functions capture lexical this; regular methods use dynamic this.

16) Closures

function counter(){
  let c = 0;
  return () => ++c; // remembers c
}
const inc = counter();
inc(); // 1, then 2, ...

17) Callbacks & Higher‑Order Functions

function withLog(fn){
  return (...args) => { console.log('call'); return fn(...args) }
}
const plus = withLog((a,b)=>a+b);

18) Promises

fetch('/api')
  .then(r => r.json())
  .then(data => console.log(data))
  .catch(console.error)
  .finally(() => console.log('done'));

19) Async / Await

async function load(){
  try {
    const res = await fetch('/api');
    const data = await res.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

26) Hoisting

27) Immutability

const state = { count: 0 };
// ❌ state.count++ (mutation)
// ✅ create a new object
const next = { ...state, count: state.count + 1 };

28) Shallow vs Deep Copy

const a = { n:1, nested:{ x:1 } };
const shallow = { ...a };                // shares nested reference
const deep = JSON.parse(JSON.stringify(a)); // naive deep copy
