JavaScript Before React - Complete Preparation Guide

This page will teach you everything you must know in JavaScript (ES6+) before you start learning React. Each topic is explained in depth with examples.

üîë JavaScript Basics (Absolute Must)

Before diving into React, you must be confident with these fundamentals. They form the building blocks of any JavaScript program.

1. Variables & Scope

Variables store data. JavaScript provides var, let, and const. The major difference is in their scope and mutability.

let count = 0;       // can be updated, block scoped
const PI = 3.14;     // cannot be reassigned
var oldWay = 10;     // function scoped (avoid in modern JS)

2. Data Types

JavaScript has primitive types such as string, number, boolean, null, undefined, symbol, bigint, and reference types such as objects, arrays, and functions.

3. Operators

Operators perform actions like math, comparisons, and logic.

  • Arithmetic: +, -, *, /
  • Comparison: === (strict equality), !==
  • Logical: &&, ||, !

4. Strings

Strings can be manipulated with methods and template literals.

const name = "Sajid";
console.log(`Hello ${name.toUpperCase()}!`);

5. Conditionals

Use if, else if, and else to execute code conditionally.

if (score >= 90) grade = 'A';
else if (score >= 75) grade = 'B';
else grade = 'C';

6. Loops

Loops allow repeating code efficiently.

for (let i = 0; i < 3; i++) {
  console.log(i);
}

7. Functions

Functions group logic into reusable blocks. Learn declaration, expression, and arrow functions.

// Declaration
function add(a, b){ return a + b; }
// Expression
const sub = function(a, b){ return a - b; }
// Arrow function
const mul = (a, b) => a * b;

üß∞ Intermediate JavaScript

Once you are comfortable with basics, you must know how to work with arrays, objects, and ES6+ features like destructuring.

Arrays

Arrays are ordered lists. Common methods include map, filter, and reduce.

const nums = [1,2,3,4];
nums.map(x => x * 2);        // [2,4,6,8]
nums.filter(x => x % 2 === 0); // [2,4]

Objects

Objects hold key-value pairs.

const user = { id: 1, name: "Aisha" };
console.log(user.name);      // dot notation
console.log(user["id"]);     // bracket notation

Destructuring & Spread

ES6 introduces concise syntax for extracting and combining data.

const { name } = user;  // destructuring
const copy = { ...user, role: "admin" }; // spread operator

‚ö° ES6+ Advanced

These modern features are heavily used in React.

Modules

// math.js
export const add = (a,b) => a+b;
export default function mul(a,b){ return a*b }

// app.js
import mul, { add } from './math.js';

Classes & Inheritance

class Person{
  constructor(name){ this.name = name }
  hello(){ return `Hello ${this.name}` }
}

Promises & Async/Await

async function loadData(){
  const res = await fetch('/api');
  const data = await res.json();
  console.log(data);
}
